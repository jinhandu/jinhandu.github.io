Static Methods

//Static Methods
//OPP 中
class Circle {
    constructor(radius) {
        this.radius = radius;
    }
    //Instance Method
    draw() {
    }

    //Static Method
    static parse(str) {
        const radius = JSON.parse(str).radius;
        return new Circle(radius);
    }
}
Circle.parse();
const circle = Circle.parse('{"radius":1}');
//We Use Static Method to create a instance
//We Use Static Method to create a utility functions that not belongs to a specific instance
//Instance是无法找到这个函数的 这个方法属于类

The This Keyword

  • 我们重新定义一个常量draw指向c.draw 这时候 this 却指向了window对象
  • Two ways of call" method call and function call
  • Method call this ⇒ object itself
  • Function call this ⇒ Window Object
  • If we use strict mode function call this ⇒ undefine
  • 当我们使用class的时候 JS会默认开启 Strict mode

Private Properties Using Symbols

  • ES6 computed property names
  • 写程序的时候 我们一般用_radius 代表这个属性是私有的 不可被外界访问
  • Symbol(); 是一个Unique Identifier 但是 每一个都是独立的 且不相同
  • 在定义私有方法的时候 我们使用[_draw] 作为函数名 来实现私有方法

Private Properties Using WeakMaps

  • 使用WeakMap 创建私有对象
  • WeakMap的本质是一个字典 字典的本质是 键值对 而其中的键是一个对象 之所以它很弱 是因为一但没有变量指向这个对象 那么JS就会释放掉这个内存空间
  • 在定义和使用这个私有对象的时候 需要使用set和get方法
