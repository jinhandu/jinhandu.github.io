Babel

Babel 是一个工具链，主要用于将ECMAScript 2015+ 版本的代码转换为向后兼容的JavaScript 语法，以便能够运行在当前和旧版本的浏览器或其他环境中。

preset 在我们使用ES6新特性的时候 每一个新的关键字 如 let 箭头语法 都对应一个plugin 使用preset我们不需要逐个安装新特性对应的plugin 可以弹性使用新特性

—save—dev we are going to install them as development dependence 开发者依赖 发布的时候 不会出现 只会存在在开发环境中
