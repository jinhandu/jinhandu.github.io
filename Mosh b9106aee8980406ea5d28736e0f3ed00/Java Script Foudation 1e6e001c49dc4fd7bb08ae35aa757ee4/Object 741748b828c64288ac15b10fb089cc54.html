Factory Functions

function createCircle(radius) {
    return {
        radius: radius,
        isVisiable: true,
        draw:function(){
            console.log('draw');
        }
    }
}

    return {
        radius,
        location: {
            x: 1,
            y: 2
        },

Value vs Reference Types

  • 产生上述结果的原因是 Reference Types的值不是存储在变量的内存当中的 而是存储于变量的内存中 变量得到的只是这个reference value的地址
let x={value:10}
let y=x;
x.value=20;
//这时候如果我们打印y.value得到的结果也是20

如果x是数组也一样

conclusion

  • Primitives(also called Value types) are copied by their value
  • Objects are copied by their reference

Enumerating Properties of an Object - 枚举对象

  • for...of can only be used on iterable like Array and maps Object is not iterable 所以我们不能用for of 来枚举Object
for (let key of circle) console.log(key);

TypeError: circle is not iterable

for (let key of Object.keys(circle)) console.log(key);
  • 每次我们用大括号的时候{} 系统会调用 Object()方法
  • const x={value:1} ⇒ const x=new Object() ⇒ const x=Object.call({},?)

Object.keys()

return the keys as a string array

radius
draw

Object.entries()

return each key-value pair as an array

each entry is an array

for (let entry of Object.entries(circle)) console.log(entry);
[ 'radius', 1 ]
[ 'draw', [Function: draw] ]

判断一个Object中是否有这个property

  • 其中的radius是字符串
if ('radius' in circle) console.log('yes');
else console.log("no");

conclusion

  • 3 ways to enumerate objets:
    • for in → instance[key]
    • for of → Object.keys(instance)
    • for of → Object.entries(instance)
  • 1 way to chek if a object has this property:
    • if ("string" in instance)

Cloning an Obejct

const circleClone = {};
for (let key in circle) {
    circleClone[key] = circle[key];
};
console.log(circleClone);
const newClone = Object.assign({}, circle);

const newClone2 = Object.assign({
    color: "yellow"
}, circle);
  • 这个方法中 第二个参数 第三个参数可以无限往后写, 而它会把这些对象中的键值对保存放在第一个空对象中
  • 第一个参数可以不是空对象
const bestClone = {...circle };
console.log(bestClone);
  • ...将会读取这个circle中所有的键值对 并且放到这个大括号中

take away:

  • Object assign方法将一个或者多个源对象的成员复制到新的对象中→我们可以用它克隆对象 或者合并对象
  • ...circle 作用是展开对象 并且把对象中所有的成员放到{}中
